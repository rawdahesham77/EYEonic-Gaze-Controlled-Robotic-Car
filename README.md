👁️ EYEonic – Gaze-Controlled Robotic Car

📌 Overview
A graduation project awarded A+, designed to enable hands-free mobility using eye movements.

The system combines Artificial Intelligence, Computer Vision, and Embedded Systems to translate eye gaze into real-time movement commands for a robotic car.

🎯 Problem Statement

Millions of people with mobility impairments face challenges in independent mobility.

Our goal was to build an affordable gaze-controlled mobility prototype.

💡 Solution

Using a webcam, OpenCV, and MediaPipe, the system detects eye movements.

Commands are sent wirelessly to an ESP32 that controls the robotic car.

⚙️ Features
Real-time eye tracking
Blink detection
Wireless communication
Obstacle detection
Hands-free control
